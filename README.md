### Proyecto Lógica: Consultas SQL
<p>
Este repositorio contiene una serie de ejercicios prácticos diseñados para afianzar los conocimientos fundamentales del uso de SQL con bases de datos relacionales.
</p>

#### Objectivos
Consolidar y practicas: 

- Uso de DBeaver para conectar, explorar y consultar la BBDD.
- Consultas básicas sobre una sola tabla (SELECT, WHERE, ORDER BY, etc.).
- Relaciones entre tablas mediante distintos tipos de JOIN.
- Subconsultas para resolver necesidades más complejas.
- Creación y uso de vistas para simplificar consultas.
- Manejo de datos temporales y funciones asociadas.
- Aplicación de buenas prácticas: alias claros, legibilidad y uso adecuado de claves primarias/foráneas.

#####  Herramientas utilizadas
- PostgreSQL para almacenar y consultar la información.
- DBeaver para ejecutar consultas y gestionar la base de datos de forma práctica y eficiente.
